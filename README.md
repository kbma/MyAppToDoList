# MyApp - Application Todo List

## Description

MyApp est une application web simple de gestion de tâches (Todo List) développée dans le cadre d'une formation sur le codage avec l'Intelligence Artificielle. Cette application permet aux utilisateurs de créer, gérer et organiser leurs tâches quotidiennes de manière intuitive.

## Fonctionnalités

- ✅ Ajouter de nouvelles tâches
- ✅ Marquer les tâches comme terminées
- ✅ Supprimer des tâches
- ✅ Filtrer les tâches (Toutes, À faire, Terminées)
- ✅ Sauvegarde automatique des tâches dans le navigateur (localStorage)
- ✅ Interface utilisateur responsive et moderne

## Installation

Aucune installation n'est requise. Il suffit d'ouvrir le fichier `index.html` dans un navigateur web moderne.

## Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur
2. Saisissez une nouvelle tâche dans le champ de texte
3. Cliquez sur "Ajouter" ou appuyez sur Entrée
4. Utilisez les boutons "V" pour marquer une tâche comme terminée
5. Utilisez les boutons "X" pour supprimer une tâche
6. Utilisez les filtres en haut pour afficher uniquement certaines tâches

## Technologies utilisées

- **HTML5** : Structure de la page
- **CSS3** : Styles et mise en page
- **JavaScript (ES6)** : Logique applicative et interactions
